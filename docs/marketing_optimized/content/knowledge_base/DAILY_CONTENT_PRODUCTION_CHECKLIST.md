# Daily Content Production Checklist

- [ ] メッセージのポイント (1つのみ)

- [ ] メインビジュアル作成 (3分)

- [ ] Canva MP4 を背景に設定
- [ ] 動画生成 (3-5分)

- [ ] エクスポート

## 配信準備 (5分)
- [ ] YouTube サムネイル＋説明欄設定
- [ ] Telegram キャプション作成
- [ ] Twitter/X ツイート作成

## 終了処理
