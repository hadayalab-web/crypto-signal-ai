## Phase 別推奨戦略

### Phase 1 (2025-11月): MVP

**選定: Whop**

**理由:**

1. 最速ローンチ（1日）
2. Discord/Telegram ネイティブ統合
3. ユーザーの当初計画に合致
4. 低リスク試行（年間¥557k）
5. 紹介アフィリエイト自動化

**却下候補と理由:**

- Gumroad: 手数料 12.9% は高すぎる（年間 ¥661k 損失）
- Stripe Billing: 開発 4-6w は MVP に不適切
- Lemonsqueezy: 機能はいいが、Whop で十分

---

### Phase 2 (2026-1月): 検証 & 移行検討

**トリガー:**

- ¥2.5M 累計利益達成
- 月次 ¥890,000+ 安定
- ユーザー数 100+ Early Adopters

**選定肢:**

**オプション A: Lemonsqueezy 中心へ移行**

- Stripe 傘下で信頼性向上
- SaaS API 強化
- 税務完全自動化
- 移行コスト: 低い

**オプション B: Stripe Billing 自社実装**

- 最安（0.7%）
- 完全制御
- スケーラビリティ最高
- 開発コスト: 高い（4-6w）

**オプション C: 複合戦略**

- Lemonsqueezy: 主力ストア（初心者向け）
- Stripe Billing: API/高度なニーズ（Expert向け）
- 柔軟性: 最高

## 変更不可ルール（今後のガイドライン）

今後、プラットフォーム選択を変更する場合は、**必ず**以下のプロセスを実行：

1. **複数プラットフォーム徹底比較**（最低 3+ 候補）
2. **ビジネス要件との照合**（マトリックス作成）
3. **段階的戦略の策定**（Phase ごと）
4. **リスク分析**（各選択肢）
5. **代替案の明示的却下理由**
6. **決定ログ記録**（DECISION_LOG.md）
7. **GitHub に commit**
