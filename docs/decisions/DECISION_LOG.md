## Decision 10: 販売プラットフォーム戦略確定（2025-11-03 11:25 JST）

**作成者**: ユーザー + AI 共同分析
**重要度**: CRITICAL
**ステータス**: 確定

### 背景

当初：Gumroad を一方的に提案（誤り）
改善：複数プラットフォーム（5候補）を徹底比較

### 決定内容

**Phase 1: Whop MVP (2025-11月)**

| 項目 | 判断 |
|------|------|
| 選定プラットフォーム | **Whop** |
| 取引手数料 | 3%（vs Gumroad 10%） |
| コミュニティ統合 | Discord/Telegram ネイティブ |
| ローンチ期間 | 1日 |
| 年間コスト | ¥557,904 |

### 却下理由

- **Gumroad**: 手数料 12.9% は高すぎる（年間 ¥661k 損失）
- **Stripe Billing**: 開発 4-6w は MVP 不適切（Phase 2 で検討）
- **Lemonsqueezy**: 機能いいが、Whop で十分（Phase 2 で移行も視野）
- **Patreon**: VIP 中心で CryptoSignal 要件に不適合

### リスク評価

- **高リスク:** Whop の認知度が Gumroad より低い
- **緩和策:** 品質とコミュニティ運営で差別化

### Phase 2 (2026-1月) トリガー

- ¥2.5M 累計利益達成
- 月次 ¥890,000+ 安定
- ユーザー数 100+ Early Adopters

### Phase 2 選定肢（未確定）

- オプション A: Lemonsqueezy 移行
- オプション B: Stripe Billing 自社実装
- オプション C: 複合戦略（両者並行）
