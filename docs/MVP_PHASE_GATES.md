- MRR ≥ ¥300,000
- 7日継続率 ≥ 35%
- CTR ≥ 15%
- 誤シグナル率 ≤ 50%
- 判定会議: 週次（月曜AM）

- 稼働SLA ≥ 99.5%
- 応答遅延 p95 ≤ 2s
- 主要機能 NPS ≥ +30

- Pro 500+ または 法人50社
